# /frontend

Здесь должен лежать веб-фронтенд приложения «Подбор свечи».

В текущем проекте основной HTML — `Подбор свечи.html` — лежит в корне репозитория
(чтобы открывать двойным кликом без локального сервера). Бэкенд раздаёт корень,
поэтому всё работает.

Если хотите соблюсти структуру `/frontend`, перенесите сюда:
- `Подбор свечи.html`
- `index.html`

И измените в `backend/server.js`:
```js
const ROOT = path.join(__dirname, "..", "frontend");
```
